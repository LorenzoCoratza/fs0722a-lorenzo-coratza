Come creare un template per i post di Mario Rossi

- pubblicato il 25-10-2020-

La prima cosa da fare è suddividere in blocchi logici la pagina,identificando le parti comuni da applicare a tutte le pagine

    Le varie aree saranno quindi suddivise in:

  • contenuti di intestazione del tag < header >
  • contenuti centrali variabili tel tag < main > (corpo principale)
  • contenuti di fine pagina comuni,da inserire net tag < footer >
  • formattazione del codice

    il testo dell'articolo dovrà essere formattato per idedntificare tutte le parti da selezionare con i CSS. Perciò i passi saranno:

  • progettare il markup assegnando ad ogni specifico testo il tag appropriato
  • scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  • applicare gli stili CSS
  • l'applicazione di stili CSS è cresciuta negli anni fino a raggiungere applicazione pressochè totale
